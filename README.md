# D&D 5e Character Sheet для Owlbear Rodeo

Интерактивный лист персонажа с полной поддержкой правил D&D 5e.  
Разработан как расширение для Owlbear Rodeo v2.

## Возможности

- Полностью настраиваемые характеристики, навыки, спасброски.
- Управление атаками с броском кубов в чат.
- Инвентарь с автоматическим подсчётом веса и грузоподъёмностью.
- Способности, черты, заметки, цели.
- Красивые карточки заклинаний с детальным описанием.
- Переключение между режимами редактирования и игры.

## Установка

1. Скачайте ZIP-архив последнего релиза.
2. Откройте Owlbear Rodeo, перейдите в Extensions → Load Extension.
3. Выберите скачанный архив.

## Использование

- Нажмите на иконку расширения на панели инструментов.
- Откроется боковая панель с листом персонажа.
- Используйте шестерёнку в правом верхнем углу для переключения режимов.
