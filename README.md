# D&D 5e Character Sheet for Owlbear Rodeo

## Сборка и установка

1. Установите зависимости:
2. npm install

text

2. Соберите расширение:
npm run build

text

3. В Owlbear Rodeo откройте **Developer Tools** (правый клик → Developer Tools), перейдите на вкладку **Extensions** и нажмите **Load unpacked**. Выберите папку `dist`.

4. В панели инструментов появится кнопка с иконкой листа. При клике открывается боковая панель с персонажем.

## Использование

- Данные автоматически сохраняются для текущего игрока и сцены.
- Переключение между режимами редактирования/игры — через шестерёнку в правом верхнем углу.
- Экспорт/импорт персонажа в JSON — там же.

## Иконки

Замените файлы в `public/icons/` на свои PNG/SVG.
